Description

Rear Window is a game based off of Alfred Hitchcock's "Rear Window". It is an eye spy game that loosely follows the events of the movie. You can view the scene outside your window and take photos of events that are happening. Once you find all the events of the day it will progress to the next scene. Within the game is a Wave Function Collapse demo that generates houses similar to the ones in the game. Both the game and the WFC demo were implemented in phaser3 and use a custom tile set made in Aseprite and Tiled.

Description

Time Rewind Platformer is a quick proof of concept of implementing a time manipulation system within the Bevy game engine. Within the game you control the white square which is able to move back and forth and jump. You can knock into other squares and push them around, however you are able to completely reverse time all the way back to where you began for all objects. Not only that but you are able to manipulate time locally for just the player or for just the objects. All of this was made with Bevy utilizing the Bevy Rapier physics crate. The entire game is compileable to WASM using trunk and playable on the browser.

Contributions

  • Solo developer
  • Integrated the Bevy Rapier physics engine
  • Created the entity components for storing past
  • Created systems for rewinding the past
  • Level Design
  • Implemented WASM compile

Description

Guarden is a wave defense garden sim where you fend off critters and grow upgrades. You start with a shovel but you can choose between a rake and two trowles as weapons as well. You begin the wave by going out the gate to your garden which transports you to a new scene where you can attack pests. Once all enemies are killed you can return to your garden to change weapon or harvest plants. However, plants take a few rounds to grow so you have to spend some time defending before you can harvest. The entire game was programmed in Phaser 3.

Description

The geocache program utilizes the leaflet map library to create a virtual geocaching game that is playable on mobile. The game will prompt for location services and when allowed will immediatly jump to your location and follow your device around. The caches are procedurally generated deterministically and you are only allowed to see the caches nearby. You can click on a cache to obtain a geocoin and click on another cache to deposit it. The game uses the flywieght, momento, and faceade patterns while also serializing a save state to local storage. The game will remember your progress and the state of the board.

Contributions

  • Solo Developer
  • Implemented the leaflet library
  • Created a flyweight system
  • Created momento system to store data to local stoarage through JSON serializing
  • Created coin gathering and depositing interface and system
  • Made the game playable on mobile
